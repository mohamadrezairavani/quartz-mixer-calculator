## 🎯 نحوه استفاده

### مرحله 1: وارد کردن داده‌های تست
- **وزن تست**: 100 گرم (یا وزن دلخواه)
- **خروجی‌های الک** (برحسب گرم):
  - 0-0.1 mm
  - 0.1-0.2 mm
  - 0.2-0.3 mm
  - 0.3-0.4 mm

### مرحله 2: تنظیم پارامترها
- **وزن تولید**: مقدار کل مورد نظر (کیلو)
- **درصد مواد**: تنظیم درصد هر ماده (مجموع = 100%)

### مرحله 3: محاسبه
کلیک بر دکمه **📐 محاسبه**

### مرحله 4: مشاهده نتایج
- **فرمول نهایی**: وزن دقیق هر ماده
- **فضای خالی**: تجزیه برای هر بازه الک
- **گزارش**: معلومات کامل

---

## 📐 فرمول‌های محاسبه

### 1. درصد هر بازه الک
```
درصد = (وزن بازه / وزن کل) × 100
```

### 2. فضای خالی
```
Void Fraction = (d_max - d_avg) / d_max
```
- `d_avg`: میانگین اندازه الک در بازه
- `d_max`: بیشترین اندازه الک (0.4 mm)

### 3. چگالی Packing
```
Packing Density = 1 - Void Fraction
```

### 4. تصحیح برای 100 کیلو
```
وزن نهایی = وزن اولیه × فاکتور تصحیح
```

## 🔧 استفاده برنامه‌ای

```python
from optimized_calculator import QuartzFormulaCalculator

# ایجاد محاسبه‌گر
calc = QuartzFormulaCalculator()

# تنظیم داده‌های تست
test_data = {
    '0-0.1': 10,
    '0.1-0.2': 25,
    '0.2-0.3': 35,
    '0.3-0.4': 30
}
calc.set_test_data(weight_grams=100, results=test_data)

# تنظیم درصد‌ها
calc.set_material_percentage('quartz', 60)
calc.set_material_percentage('resin', 15)
# ... و مابقی مواد

# دریافت گزارش
report = calc.get_detailed_report()

# صادرات
calc.export_json('formula.json')
```
