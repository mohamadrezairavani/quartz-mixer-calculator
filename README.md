## 📊 فرمول‌های استفاده شده

### **1️⃣ درصد هر بازه:**
```
درصد = (وزن بازه / وزن کل) × 100
```

### **2️⃣ فضای خالی:**
```
Void Fraction = (d_max - d_avg) / d_max
```

جایی که:
- `d_avg` = میانگین اندازه الک در این بازه
- `d_max` = بیشترین اندازه الک (0.4 mm)

### **3️⃣ چگالی Packing:**
```
Packing Density = 1 - Void Fraction
```

### **4️⃣ وزن مورد نیاز برای 100 کیلو:**
```
Weight = (Percentage × 100 kg) / (1 - Void Fraction)
```
