# 🏭 محاسبه‌گر میکسر کوارتز | Quartz Mixer Calculator

پروژه‌ای برای محاسبه فضای خالی بین دانه‌بندی و ترکیب مواد برای تولید 100 کیلو سنگ کوارتز

## 📋 ویژگی‌ها

✅ محاسبه فضای خالی برای 5 سطح الک مختلف:
- 0 میلی‌متر
- 0.1 میلی‌متر
- 0.2 میلی‌متر
- 0.3 میلی‌متر
- 0.4 میلی‌متر

✅ محاسبه چگالی Packing برای هر سطح

✅ فرمول کامل ترکیب مواد برای 100 کیلو:
- **کوارتز (Quartz)**: 70 کیلوگرم
- **دوده (Carbon Black)**: 15 کیلوگرم
- **پراکسید (Peroxide)**: 10 کیلوگرم
- **رزین (Resin)**: 5 کیلوگرم

## 📐 فرمول‌های استفاده شده

### فضای خالی (Void Fraction)
```
Void Fraction = (d_max - d) / d_max
```
جایی که:
- `d_max` = بیشترین اندازه الک (0.4 میلی‌متر)
- `d` = اندازه الک فعلی

### چگالی Packing
```
Packing Density = 1 - Void Fraction
```

### وزن هر ماده برای 100 کیلو
```
Weight = (Percentage / 100) × 100 kg
```

## 🔧 توسعه بیشتر

برای افزودن ویژگی‌های جدید:

1. **سفارشی‌سازی درصد مواد** - تغییر مقادیر در متغیر `materials`
2. **اضافه کردن سطح‌های الک جدید** - افزودن مقادیر به `sieve_sizes`
3. **محاسبه برای وزن‌های مختلف** - تغییر `batch_size`
4. **صادرات به فرمت‌های مختلف** - CSV، Excel، PDF
